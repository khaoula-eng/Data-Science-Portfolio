# ❤️ Heart Disease Classification

## 📌 Project Overview

This project focuses on building a machine learning classification model to predict whether a patient has heart disease based on medical and health-related features.

The goal of this project is to explore the dataset, understand the relationships between different health indicators, train classification models, and evaluate their performance using appropriate machine learning metrics.

This project is part of my **Data Science Portfolio** and demonstrates my ability to work on a complete machine learning workflow, from data exploration to model evaluation.

---

## 🎯 Project Objective

The main objective is to answer the following question:

> Can we predict the presence of heart disease using patient health data?

To achieve this, the project includes:

- Data loading and exploration
- Exploratory Data Analysis
- Data visualization
- Feature and target separation
- Model training
- Model comparison
- Model evaluation using classification metrics

---

## 🗃️ Dataset Description

The dataset contains patient medical information related to heart disease diagnosis.

The target variable is:

| Column | Description |
|------|-------------|
| `target` | Indicates whether heart disease is present or not |

Main features include:

| Column | Description |
|------|-------------|
| `age` | Age of the patient |
| `sex` | Gender of the patient |
| `cp` | Chest pain type |
| `trestbps` | Resting blood pressure |
| `chol` | Serum cholesterol level |
| `fbs` | Fasting blood sugar |
| `restecg` | Resting electrocardiographic results |
| `thalach` | Maximum heart rate achieved |
| `exang` | Exercise-induced angina |
| `oldpeak` | ST depression induced by exercise |
| `slope` | Slope of the peak exercise ST segment |
| `ca` | Number of major vessels colored by fluoroscopy |
| `thal` | Thalassemia result |
| `target` | Heart disease diagnosis |
